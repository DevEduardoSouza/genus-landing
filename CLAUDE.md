# Genus Tech Landing Page

## Stack
- Next.js 15 (App Router, TypeScript)
- GSAP (ScrollTrigger, animações)
- Custom CSS (unidades vw, sem Tailwind)
- Fonte Space Grotesk (Google Fonts)

## Design System
- Estética brutalista-editorial inspirada no carlesfaus.com
- Background branco (#ffffff), texto preto (#000000)
- mix-blend-mode: exclusion na nav/footer
- Tipografia toda em uppercase
- Espaçamentos e font-sizes em vw
- Links com colchetes: [Sobre] [Projetos] [Contato]
- Cor de destaque: #68A063 (verde Genus, usado pontualmente)

## Estrutura
- `/app` - Páginas (home, sobre, projetos, contato)
- `/components` - Componentes reutilizáveis
- `/hooks` - Hooks customizados React
- `/lib` - Utilitários e configs
- `/styles` - CSS por componente/página
